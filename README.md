# 📌 Project Overview

This project is a personal portfolio website created using **HTML and CSS**.
The website presents my personal information, technical skills, and includes a contact form for user interaction.

The portfolio demonstrates the use of **HTML5 semantic elements** along with **CSS styling techniques** to create a visually appealing and responsive design.

---

# 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling, layout, and responsiveness
